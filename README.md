# Java-Projects
<h5>1. Word Counter</h5>
The user should enter either a sentence or a paragraph of English words.<br>
 For that we input word String using String datatype. We trim the unwanted whitespace if any present inside the entered String.<br>
We find out by starting character of each word after end of whitespaces using loop and increment the count after occurrence of new word starting character.<br>
This way, after we get count of words in String we print it on IDE terminal.<br>

<h5>2.Time Adder Calculator</h5>
In this project we input the current time as “Starting Time” in the Hour-Minute-Second format. Also, the time to go forward by count is input in the same H-M-S format eg. 5 hours 20 minutes ahead.<br>
Here we consider calculation as per 24 hour format.<br>
This code finds out actual forward time,  when a required time gap is added to mentioned current time.
For each minute after 60 seconds are over, there should be addition of carry for appending time ahead. Same goes for each hour after 60 minutes, gets carry appended.<br>
 For next day, we subtract value from total 24 hours and for the same day, we directly mention the added time.<br>
We display both start time and time ahead in IDE terminal window.<br>
